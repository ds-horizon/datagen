---
title: MySQL Sink Configuration
---

A MySQL sink config defines how datagen connects and writes data to MySQL.

### Example
```json
{
  "sink_name": "pluto_mysql",
  "sink_type": "mysql",
  "config": {
    "host": "localhost",
    "database": "datagen",
    "port": 3306,
    "username": "dg",
    "password": "dg",
    "batch_size": 1,
    "throttle": "10ms"
  }
}
```

### Config fields

| Field       | Type    | Description                                   | Required | Default |
|-------------|---------|-----------------------------------------------|----------|---------|
| host        | string  | MySQL server hostname or IP                   | Yes      | -       |
| database    | string  | Database name to write into                   | Yes      | -       |
| port        | number  | MySQL port                                    | No       | 3306    |
| username    | string  | Database user                                 | Yes      | -       |
| password    | string  | Database password                             | Yes      | -       |
| batch_size  | number  | Records per batch insert                      | No       | 1       |
| throttle    | string  | Delay between batches (e.g., "10ms", "1s")    | No       | -       |

**Notes:**
- Ensure user has INSERT privileges on target tables
- Use appropriate `batch_size` and `throttle` to control load rate
